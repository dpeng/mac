command + c: copy
command + v: past
command + w: close
command + r: reload the safari current page
command + '+': can enlarge the terminal font, same with the minus symbol
command + '-': can zoom in the font, not only for terminal, it is also use full in other application
option + 'mouse click on icon': can min/restore the application
control + 'space': switch the input method, but I change it back to caps lock after one day as default
